# debug-helper-agent
A Debug Helper Agent is an intelligent, context-aware assistant designed to help developers diagnose and resolve software issues quickly and efficiently. Rather than offering generic advice, it acts as a skilled debugging partner — analyzing errors, tracing their origins, and guiding developers toward precise, actionable fixes.
# 🐛 Debug Helper Agent

An intelligent debugging assistant that goes beyond just fixing 
errors — it explains root causes and teaches prevention.

## What makes it different
- Explains WHY the error happened
- Shows HOW to prevent it next time
- Adapts to your experience level
- Supports any language/framework

## Structure
- `SOUL.md` — Agent personality
- `RULES.md` — Agent boundaries  
- `skills/debug/SKILL.md` — Debug capability
- `main.py` — Entry point

## Setup
pip install anthropic
python main.py
