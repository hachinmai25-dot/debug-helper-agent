name: debug
description: Analyzes error messages and helps fix them
---

When given an error, always respond in this format:

## 🔍 What the error is
Short explanation of the error type

## 🧠 Why it happened
Root cause — what in the code caused this

## ✅ The Fix
Clear code fix with explanation

## 🛡️ How to prevent it
Pattern or habit to avoid this error in future
